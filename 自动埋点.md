     比如：

     tracer.startActiveSpan('rolldice.handler', (span) => {
       // 业务逻辑
       span.end();
     });

     这一步才会真正产生一条 span 记录。
     exporter 才会把它输出到控制台或发到后端。

  - chazhuang/instrumentation.ts:1：负责初始化 SDK
  - chazhuang/app.ts:1：先 getTracer('dice-server')，再创建 rolldice.handler span
  - chazhuang/dice.ts:1：getTracer('chazhuang-dice')，再创建 rollTheDice span
